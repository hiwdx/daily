标题：Anthropic 推出 Claude Managed Agents 企业沙箱和 Project Glasswing 网络安全计划

来源：Anthropic News · 2026年05月27日（昨日）

摘要：

  • 发生了什么： Anthropic 宣布 Claude Managed Agents 可在企业控制的沙箱中运行，并连接到用户的私有 Model Context Protocol (MCP) 服务器，使 agent 执行工具的环境和访问的服务都在企业边界内运行；同时 Project Glasswing 扩展到 Claude Security 公开测试版，面向符合条件的安全团队提供新的网络验证工具，帮助扫描代码库、分类漏洞并生成修复。

  • 对谁有影响： 包括 Spotify、Base44、Legora 等企业客户已在使用 Claude Code 进行优化和部署。

产品技术视角：Agent 框架向企业实际部署转进，MCP 和隔离沙箱使 AI 系统可在受控环境中安全操作；AI 模型在大规模漏洞发现上的超人能力正式化为企业安全工具。

标题：Verge Labs 作为边界型 AI 实验室推出，基于十年脑组织数据构建疾病生物学基础模型

来源：PR Newswire · 2026年05月27日

摘要：

  • 发生了什么： Verge Labs（原 Verge Genomics）推出为边界型 AI 实验室，基于十年的专有脑组织数据构建人类疾病生物学的基础模型。
  • 为什么重要： VergeDB 数据库包含超 12,000 个脑转录组数据（来自 6,000 名患者）、1,500 万个单细胞 profile，以及配套的蛋白质组学、基因组学和临床数据

构建了"疾病生物学的 world model"——一个生成性模型，模拟疾病在患者体内如何演进，直接基于人类组织而非动物或细胞替代品训练。 - 对谁有影响： 公司首个 AI 发现的资产在 2025 年末完成 Phase 1b 试验，其临床、生物标志物和患者数据已被纳入 VergeDB，成为模型的核心训练信号。

产品技术视角：将 AI 世界模型从虚拟环境应用到真实生物学数据，用 foundation model 重新定义药物研发的假设生成逻辑。
