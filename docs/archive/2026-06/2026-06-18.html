标题： OpenAI 推出 Deployment Simulation 功能，在新模型发布前通过回放历史对话进行评估

来源：LLM Stats · 2026-06-16

摘要：

OpenAI 在 6 月 16 日推出 Deployment Simulation，该方法通过回放过去的对话到新候选模型中，然后评估完成情况以估计部署时的表现。

  • 这是模型评估从静态基准到动态真实对话的重要转变，降低生产环节的部署风险。
  • 对依赖 OpenAI API 的开发者和企业，可更安心地升级到新模型版本。

产品技术视角：从离线 benchmark 评估转向在线回放真实对话数据的评估方案，彰显了模型安全部署工程的成熟度提升。

标题： Inception 发布 Mercury 2 扩散架构推理语言模型，实现超 1000 tokens/秒的推理速度

来源：dentro.de · 2026-06-17（约当日数据）

摘要：

Inception 推出的 Mercury 2 采用扩散架构实现并行生成，针对 agent 循环和实时语音交互等生产场景，在低延迟约束下实现推理级别的质量。

  • 打破了传统自回归模型的延迟瓶颈，为 agent 工作流提供了新的架构选择。
  • 对实时对话系统、智能体应用、流式处理等场景有重大意义。

产品技术视角：扩散模型在 LLM 推理中的首次大规模应用，同时实现高质量和高速度的平衡，代表了序列生成架构的创新突破。
